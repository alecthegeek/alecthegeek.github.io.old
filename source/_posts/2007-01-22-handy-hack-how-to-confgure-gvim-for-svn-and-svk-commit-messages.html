'Handy Hack: How to configure gvim for svn and svk commit messages' tags: - Linux - Open Source Software - Software Configuration Management - Work Practices status: publish type: post published: true meta: {} --- If you use svn or svk and would like to use the GUI version of Vim to edit your commit messages then add the following in your .profile, SVN_EDITOR="$EDITOR --nofork"; export SVN_EDITOR. This of course assumes that you already have have $EDITOR set to some value such as /usr/bin/gvim. If you use a csh style shell then modify to suite.

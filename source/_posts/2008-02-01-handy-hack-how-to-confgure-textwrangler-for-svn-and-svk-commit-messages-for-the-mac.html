'Handy Hack: How to configure TextWrangler for svn & svk commit messages on OS/X' tags: - Mac - Software Configuration Management - Work Practices status: publish type: post published: true meta: {} --- In a similar fashion to a previous post about gvim you can use TextWrangler as your SVN or SVN log message editor by setting $SVN_EDITOR to the value of "/usr/bin/edit -w" and install the TextWrangler command line tools. I'm guessing that something similar would also work for BBEdit as well.

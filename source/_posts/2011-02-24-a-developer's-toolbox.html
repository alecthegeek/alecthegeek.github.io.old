--- layout: post title: A developer's toolbox tags: [] status: draft type: post published: false meta: _edit_last: '451289' --- What tools does a developer need?
  • A decent editor. Vim or Emacs if you are not sure
  • A proper DVCS -- i.e. Git
  • An SQL database. SQLite is a nice simple option
  • A proper shell environment -- e.g. Linux, OS/X or Cygwin in Windows (at a pinch)

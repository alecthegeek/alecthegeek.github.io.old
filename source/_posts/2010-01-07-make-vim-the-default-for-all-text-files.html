--- layout: post title: ! 'Handy Hack: Make Vim the default editor for all text files' tags: - Windows status: publish type: post published: true meta: _edit_last: '451289' --- I just added the following tip to the Vim Tips wiki
To make Vim the editor for all text file types, as defined by MS Windows, try from the Windows command prompt: 
ftype txtfile="C:\Program Files\Vim\vim72\gvim.exe" --remote-tab-silent "%1"

First you need to install the Python Image Library and QR Code library. On my OS X system that was easy ``` bash $ pip install qrcode ``` Then you can create the following simple Pyton script. ``` python #! /usr/bin/env python import sys import qrcode qrcode.make("".join(sys.stdin.readlines())).save("/dev/stdout","png") ``` It reads any text from it's standard input and writes a graphic image to the standard output. So you can do this ``` bash $ echo "My QR Text"|makeQR > myqr.png ```

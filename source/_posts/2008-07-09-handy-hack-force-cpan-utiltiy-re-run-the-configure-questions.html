'Handy Hack: Force cpan utiltiy re-run the configure questions' tags: - Perl status: publish type: post published: true meta: _edit_last: '451289' --- When you first the cpan command line utility it asks you a rather tedious list of questions about your setup. Occasionally you need to go through it all again. Use the command 'o conf init' at the cpan prompt to force this behaviour.
